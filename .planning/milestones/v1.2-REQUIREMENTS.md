## Out of Scope (v1.2)

- OpenAI-generated explanation text for score breakdown — introduces latency, burns budget, creates formula-prompt drift. Deterministic template only.
- Spotify playback state tracking — deprecated for third-party apps as of Feb 2026.
- `compound = AND` (was_liked AND was_saved) — too sparse for single-user daily-gem app; OR semantics chosen.

## Deferred Beyond v1.2

- **Full-cascade feedback filtering** — AI classifies track traits from text → exclude track/album/artist/genre-mood by confidence. Scoped into the v1.2 goal but not shipped in Phase 10; only the sentiment-sync slice landed.
- Evaluation dashboard — learning curve, per-source win rates, A/B framework (next milestone).
- Audio feature proxy via AcousticBrainz/Last.fm.
- Collaborative filtering + Postgres migration.
- Production deployment.
