## Spacing Scale

Declared values (multiples of 4 only). Derived from existing component patterns (MetricsStrip, DailyGem, TopArtists).

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, genre tag inline padding (`px-2.5 py-0.5`) |
| sm | 8px | Gap between stat items within a row (`gap-8` is 32px — see exception) |
| md | 16px | Default element spacing, section padding inner |
| lg | 24px | Card internal padding (`p-6`), section top/bottom padding |
| xl | 32px | Layout column gap (`gap-8`), flex row gap between stat groups |
| 2xl | 48px | Vertical rhythm between chart section sub-blocks |
| 3xl | 64px | Profile page section vertical padding (`py-64` = 256px — see exception) |

Exceptions:
- Chart section wraps in `py-16 px-4 md:px-8 lg:px-16` to match MetricsStrip horizontal padding pattern exactly.
- MetricsStrip uses `gap-8` (32px) between stat items — carry this pattern into the new section.
- Profile page hero section uses `py-64` (256px) for full-viewport scroll effect — new bottom section does NOT replicate this; use `py-16` at most.

## Color

Extracted from tailwind.config.ts and layout.tsx. Phase 4 components must not introduce new colors.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#0c0a09` (`bg-stone-950`) | Page background — body-level, set in layout |
| Secondary (30%) | `#111827` (`bg-gray-900`) | Card/panel backgrounds (TopArtists, new chart section container) |
| Secondary-alt | `#1f2937` (`bg-gray-800`) | Nested card backgrounds, hover states |
| Accent (10%) | `#1DB954` (`text-green` / `bg-green`) | Reserved list below |
| Destructive | `#f87171` (`text-red-400`) | Error states only — no destructive actions in this phase |

Accent `#1DB954` is reserved for:
1. Recharts line stroke on the 7-day like-rate trend chart
2. Recharts bar fill on the taste-profile genre bar chart
3. Stat values in the improvement story (e.g., "Now: 71%")
4. Section divider accent (left-border on blockquote pattern, if used)

Do NOT use the accent for: axis labels, grid lines, background fills, section headings, or genre tag borders.

Recharts grid lines: `stroke="#374151"` (gray-700). Recharts tooltip background: `bg="#111827"` (gray-900), border `#374151` (gray-700), text `#f3f4f6` (gray-100).

## Component Inventory

New components to create (all in `frontend/app/profile/components/`). Pattern: `<Name>/<Name>.tsx` with `"use client"` directive, using `get()` from `@/services/axios`.

### 1. `LikeTrendChart/LikeTrendChart.tsx`

Recharts `LineChart` (responsive, via `ResponsiveContainer`).

- Data source: `GET /api/recommendation-trend/` — returns array of `{date: string, like_rate: number}` points
- X-axis: `date` field, formatted as `MMM D` (e.g. "May 5"). Use `tickFormatter` with `new Date(date).toLocaleDateString('en-US', {month:'short', day:'numeric'})`.
- Y-axis: 0–100, label "Like rate (%)", tick interval 25
- Line: `stroke="#1DB954"`, `strokeWidth={2}`, `dot={false}`, `type="monotone"`
- No legend. Tooltip shows date + like_rate value (e.g. "May 5 — 67%").
- Container: `width="100%"` `height={220}`
- Cold-start (<2 data points): render empty-state message (see Copywriting Contract below) instead of chart

### 2. `TasteProfileChart/TasteProfileChart.tsx`

Recharts `BarChart` (horizontal, `layout="vertical"`).

- Data source: `GET /api/recommendation-metrics/` — `top_genres` field from existing MetricsStrip interface; extend with `top_genres_pct: Array<{genre: string, pct: number}>` from same endpoint (separate field, does not break existing interface — see D-02 note below).
- Y-axis: genre names, `width={120}`, `tick={{fill:"#9ca3af", fontSize:12}}` (gray-400)
- X-axis: 0–100, tick at 0/25/50/75/100, label hidden, `tick={{fill:"#6b7280", fontSize:12}}`
- Bar: `fill="#1DB954"`, `radius={[0,3,3,0]}`
- Shows top 10 genres, normalized to percentage (genre_count / total * 100)
- Container: `width="100%"` `height={280}`
- Cold-start (no taste vector / empty genres): render empty-state message instead of chart

D-02 interface extension note: The existing `top_genres: string[]` field stays intact for MetricsStrip. Add a parallel `top_genres_pct: Array<{genre: string, pct: number}>` field to the same endpoint response. MetricsStrip ignores unknown fields — no breaking change.

### 3. `ImprovementStory/ImprovementStory.tsx`

Inline comparison block — no chart, pure text/stat layout.

- Data source: `GET /api/recommendation-metrics/` — add `improvement_story: {first_7_rate: number|null, last_7_rate: number|null, delta: number|null}` to response.
- Layout: two side-by-side `Stat`-style blocks (same pattern as MetricsStrip `Stat` component)
  - Left: "When I started" / `first_7_rate` formatted as percentage
  - Right: "Now" / `last_7_rate` formatted as percentage, value in `text-green`
  - Delta arrow if positive: `▲ +{delta}pp` in `text-green text-xs`; if negative: `▼ {delta}pp` in `text-red-400 text-xs`
- Cold-start (fewer than 2 gems): show skip — render nothing (do not show empty state, just omit the block)

### 4. `DiversityScore/DiversityScore.tsx`

Single stat display — one number with label and tooltip explanation.

- Data source: `GET /api/recommendation-metrics/` — add `diversity_score: number|null` (0.0–1.0, mean pairwise Jaccard distance).
- Display: `{(diversity_score * 100).toFixed(0)}%` in `text-green text-lg font-bold tabular-nums`
- Label below: "Genre diversity" in `text-xs text-gray-500 uppercase tracking-widest`
- Tooltip (hover/focus): "How different your recommended tracks have been from each other. 100% = every track from a completely different genre."
- Placed in the same bottom section, alongside the stat numbers — matches MetricsStrip `Stat` component layout.
- Cold-start (null): show "—" instead of percentage.

## Interaction States

All new components are display-only (no destructive actions in this phase).

| State | Behaviour |
|-------|-----------|
| Loading | Show three bouncing dots (same pattern as DailyGem: `w-2 h-2 rounded-full bg-green animate-bounce`) with staggered delay, minimum 40px height placeholder |
| Data present | Render chart / stat as specified above |
| Cold-start / empty | Render empty-state copy (see Copywriting Contract) in place of chart |
| API error | Render error copy (see Copywriting Contract); no retry button (charts are informational, matching MetricsStrip silent-fail pattern) |
| Recharts tooltip | Visible on hover; keyboard-inaccessible is acceptable (charts are supplementary, not primary data path) |
